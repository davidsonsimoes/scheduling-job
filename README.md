# Scheduling Job

[![N|Solid](http://davidsonsimoes.com.br/images/logo2.png)](http://davidsonsimoes.com.br/)

Solução utilizada

  - Node.js
  - TypeScript

### Rodando o Script

Necessário o [Node.js](https://nodejs.org/) para rodar.

Comandos:

```sh
$ git clone git@github.com:davidsonsimoes/scheduling-job.git
$ cd scheduling-job
$ yarn
$ yarn start
```
